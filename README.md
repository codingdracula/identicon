# Identicon

## Version Information
```
app: :identicon,
version: "0.1.0",
elixir: "~> 1.7",
```
## Install elixir
> https://elixir-lang.org/install.html

## How to test
```
mix test
```

## How to run

```
iex -S mix
Identicon.transformation("name")
```

> Images by default are saved to the images folder.
